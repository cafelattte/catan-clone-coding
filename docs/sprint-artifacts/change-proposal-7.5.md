## 1. Change Summary

게임 로직(GameState)과 UI(game.lua)를 연결하여 실제 플레이 가능한 게임으로 통합.

### Problem Statement

Epic 7의 기존 스토리들(7.1-7.4)로 다음이 구현됨:
- 턴 순서 관리 (GameState)
- 페이즈 관리 (GameState)
- 메인 메뉴 씬 (menu.lua)
- 게임 종료 씬 (game_over.lua)

하지만 **game.lua가 GameState 메서드를 호출하지 않음**:
- 주사위 굴리기 UI 없음
- 건설 실행 없음 (선택만 가능, Actions 모듈 미연결)
- 턴 종료 UI 없음

### Solution

Story 7.5 "게임 플레이 통합" 추가하여 game.lua에 액션 버튼 패널 추가 및 GameState 메서드 연결.
